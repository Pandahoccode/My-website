# Planificateur de Voyage Interactif pour Lille

Développement d'une application interactive de planification de voyage pour Lille utilisant Streamlit, intégrant l'analyse de données et les API de transport en temps réel.

## Capacités

- **Optimisation des Coûts** : Optimisation des coûts de transport via les API SNCF et OpenRouteService pour une comparaison aisée entre le train et la voiture.
- **Itinéraires Personnalisés** : Génération automatique d'itinéraires personnalisés basés sur le budget, les dates et le nombre de voyageurs.
- **Informations Interactives** : Affichage interactif d'informations locales (hôtels, météo, restaurants) sur des cartes géographiques.
- **Interface Bilingue** : Interface simple et bilingue avec un tableau de bord dynamique pour gérer tous les paramètres du voyage.
