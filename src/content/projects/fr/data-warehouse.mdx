# Entrepôt de Données Chinook avec ODI

Projet universitaire transformant la base de données Chinook (11 tables) en un entrepôt de données en schéma en étoile (réduit à 7 tables) utilisant Oracle Data Integrator (ODI) pour analyser les ventes de musique.

## Étapes Techniques Clés

- **Automatisation ETL** : Développement d'un pipeline ETL automatisé pour extraire, transformer et charger les données de 11 tables sources vers 7 tables dans l'entrepôt de données.
- **Schéma en Étoile** : Conception d'un schéma en étoile optimisé (dimensions + faits) pour des requêtes d'intelligence d'affaires rapides et efficaces.
- **Centralisation** : Mise en place d'un entrepôt de données centralisé permettant des analyses claires sur les ventes.
- **Maîtrise des Outils** : Utilisation professionnelle d'Oracle Data Integrator (ODI) pour la modélisation des données et la gestion complète du projet.
