# Lille Interactive Travel Planner

Development of an interactive travel planning application for Lille using Streamlit, integrating data analysis and real-time transport APIs.

## Capabilities

- **Cost Optimization**: Optimization of transport costs through SNCF and OpenRouteService APIs for easy comparison between train and car.
- **Customized Itineraries**: Automatic generation of customized itineraries based on budget, dates, and number of travelers.
- **Interactive Information**: Interactive display of local information (hotels, weather, restaurants) on geographic maps.
- **Bilingual Interface**: Simple and bilingual interface with a dynamic dashboard for managing all travel parameters.
